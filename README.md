# 🚀 Full-Stack Web Development & Data Analytics Project

## 📌 Project Overview
This repository contains the tasks, assignments, and projects completed during my internship. The project showcases a comprehensive blend of Full-Stack Web Development using Node.js/Express and Data Science/Analytics using Python.

---

## 📁 Repository Structure & Modules

### 1. 🌐 Web Development & Backend (Node.js / Express)
The repository features a structured backend application designed for scalability:
* *landing page/*: The frontend user interface of the application.
* *config/*: Configuration files for database connections and environment setups.
* *models/*: Database schemas and data models defining the structure of the data.
* *routes/*: API endpoints handling the application's business logic and routing.
* *middleware/*: Functions for request preprocessing, validation, and security controls.
* *server.js*: The main entry point that initializes and runs the backend server.
* *package.json*: Contains metadata, scripts, and npm dependencies required for the project.

### 2. 📊 Data Science & Machine Learning Tasks
This section involves data processing, cleaning, and analysis using Python and Jupyter Notebooks:
* *Online Retail.xlsx & Online Retail_clean.csv*: Datasets used for exploring transaction data and running retail analytics.
* *WA_Fn-UseC_-Telco-Customer-Churn.csv*: A popular industry dataset utilized for customer behavior analysis and churn prediction modeling.
* *task1.ipynb, task2.ipynb, task3.ipynb*: Interactive Jupyter Notebooks containing the step-by-step process for data cleaning, data visualization, and exploratory data analysis (EDA).

### 3. 🐍 Python Programming Assignments
A collection of foundational Python scripts focusing on logic-building and problem-solving:
* *armstrong.py*: A program to check whether a given number is an Armstrong number.
* *factorial.py*: Calculates the factorial of a specified number.
* *fibonaci.py*: Script to generate and display the Fibonacci sequence.
* *leep_year.py*: Checks if a given year is a leap year.
* *palandrom.py*: Validates whether a string or number is a palindrome.
* *reverse_string.py*: Demonstrates string manipulation by reversing text.
* *odd_even.py & sum.py*: Scripts handling basic arithmetic operations and conditional statements.
